## 시스템 소개

### 이 시스템으로 무엇을 할 수 있나요?

이 시스템은 노인 일자리 사업 참여자를 효율적으로 관리하기 위한 웹 기반 관리 도구입니다.

#### 주요 기능

1. **참여자 정보 관리**
   - Excel 파일로 여러 참여자의 정보를 한 번에 등록
   - 참여자 이름, 전화번호, 소속 사업단 정보 관리

2. **활동 장소 관리**
   - Excel 파일로 여러 활동 장소를 한 번에 등록
   - 주소를 입력하면 지도에 자동으로 표시
   - 여러 장소를 한눈에 지도에서 확인

3. **대기 번호표 시스템**
   - 참여자들이 순서를 기다릴 때 사용하는 번호표 발급
   - 현재 호출 번호를 실시간으로 업데이트
   - 참여자가 본인 번호를 스마트폰으로 확인 가능

4. **출석 관리** (향후 활용 예정)
   - 참여자별 출석 기록 관리
   - 장소별 일정 관리

## 시스템 접속 방법

### 인터넷 브라우저로 접속하기

1. **컴퓨터나 스마트폰에서 인터넷 브라우저를 실행합니다**
   - 크롬(Chrome), 엣지(Edge), 사파리(Safari) 등 아무 브라우저나 사용 가능합니다

2. **주소창에 시스템 주소를 입력합니다**
   ```
   http://시스템주소:5173
   ```
   *(실제 주소는 시스템 관리자에게 문의하세요)*

3. **메인 화면이 나타나면 접속 완료!**

## 주요 기능 안내

시스템 메인 화면에서 다음 메뉴들을 볼 수 있습니다:

### 📍 Excel 관리 (지도)
- **용도**: 활동 장소 정보를 Excel로 등록하고 지도에서 확인
- **사용 시점**: 새로운 활동 장소를 추가하거나 기존 장소를 확인할 때

### 👥 회원 Excel 업로드
- **용도**: 참여자 정보를 Excel로 한 번에 등록
- **사용 시점**: 신규 참여자를 등록하거나 기존 정보를 업데이트할 때

### 📋 번호표 시스템 (방 목록)
- **용도**: 대기 번호표를 발급하고 관리
- **사용 시점**: 참여자들이 순서를 기다려야 할 때

### 🔐 방 관리 (관리자)
- **용도**: 번호표 방을 만들고 현재 호출 번호를 변경
- **사용 시점**: 새로운 대기실을 만들거나 번호를 호출할 때

## 기능별 사용 방법

### 1️⃣ 참여자 정보 Excel로 등록하기

#### 필요한 것
- 참여자 정보가 정리된 Excel 파일

#### Excel 파일 준비하기

Excel 파일은 다음과 같은 형식으로 준비해주세요:

| 이름 | 전화번호 | 사업단 |
|------|---------|--------|
| 홍길동 | 010-1234-5678 | 공원관리단 |
| 김영희 | 010-2345-6789 | 환경미화단 |
| 이철수 | 010-3456-7890 | 공원관리단 |

**중요 사항:**
- 첫 번째 줄은 제목(이름, 전화번호, 사업단)으로 작성
- 두 번째 줄부터 실제 참여자 정보를 입력
- 모든 정보를 빠짐없이 입력해주세요

#### 단계별 사용 방법

1. **메인 화면에서 "회원 Excel 업로드" 메뉴 클릭**

2. **"엑셀 파일을 선택하세요" 버튼 클릭**
   - 파일 선택 창이 열립니다
   - 준비한 Excel 파일을 찾아서 선택
   - "열기" 버튼 클릭

3. **업로드된 정보 확인**
   - 화면에 표 형식으로 참여자 정보가 표시됩니다
   - 번호, 사업단, 이름, 전화번호가 맞는지 확인하세요
   - 잘못된 정보가 있다면 Excel 파일을 수정 후 다시 업로드

4. **"데이터베이스에 저장" 버튼 클릭**
   - 참여자 정보가 시스템에 저장됩니다
   - "○명의 회원 정보가 성공적으로 저장되었습니다" 메시지가 나타나면 완료!

#### 💡 팁
- Excel 파일은 .xlsx 또는 .xls 형식만 가능합니다
- 파일을 잘못 선택했다면 다시 파일을 선택하면 됩니다

---

### 2️⃣ 활동 장소 Excel로 등록하고 지도에서 보기

#### 필요한 것
- 활동 장소 정보가 정리된 Excel 파일

#### Excel 파일 준비하기

| 사업단명 | 주소 |
|----------|------|
| 공원관리단 | 서울특별시 강남구 테헤란로 123 |
| 환경미화단 | 서울특별시 서초구 서초대로 456 |

#### 단계별 사용 방법

1. **메인 화면에서 "Excel 관리 (지도)" 메뉴 클릭**

2. **Excel 파일 선택 및 업로드**
   - "엑셀 파일을 선택하세요" 버튼 클릭
   - 준비한 Excel 파일 선택
   - 자동으로 파일이 업로드됩니다

3. **지도에서 확인하기**
   - 업로드된 장소들이 지도에 숫자 마커로 표시됩니다
   - 마커를 클릭하면 상세 정보를 볼 수 있습니다
   - 지도를 확대/축소하며 위치를 확인하세요

4. **정보가 맞다면 "데이터베이스에 저장" 버튼 클릭**
   - 장소 정보가 시스템에 저장됩니다

#### 💡 팁
- 주소는 정확하게 입력해야 지도에 제대로 표시됩니다
- "서울시" 대신 "서울특별시"처럼 공식 주소를 사용하세요
- 도로명 주소 또는 지번 주소 모두 사용 가능합니다

---

### 3️⃣ 번호표 시스템 사용하기

번호표 시스템은 두 가지 역할로 나뉩니다:
- **관리자**: 대기실(방)을 만들고 번호를 호출
- **참여자**: 번호표를 받고 본인 차례를 확인

#### A. 관리자 - 대기실 만들기

1. **메인 화면에서 "🔐 방 관리 (관리자)" 메뉴 클릭**

2. **새 방 만들기**
   - 화면 상단에 방 제목 입력란이 있습니다
   - 예: "상담 대기실", "서류 접수 대기"
   - "방 만들기" 버튼 클릭

3. **방이 생성되었습니다!**
   - 화면에 방 목록이 표시됩니다
   - 각 방의 정보가 표시됩니다:
     - 방 제목
     - 현재 호출 번호
     - 대기 인원
     - 방 입장 링크

#### B. 관리자 - 번호 호출하기

1. **방 관리 화면에서 해당 방의 "입장" 버튼 클릭**

2. **현재 번호 변경하기**
   - "다음 번호 호출" 버튼 클릭 시: 현재 번호가 1씩 증가
   - 또는 직접 번호를 입력하고 "번호 변경" 버튼 클릭

3. **변경된 번호가 실시간으로 참여자 화면에 표시됩니다**

#### C. 참여자 - 번호표 받기

1. **참여자에게 방 입장 링크를 안내합니다**
   - 관리자 화면에서 "링크 복사" 버튼을 클릭하여 복사
   - 카카오톡이나 문자로 참여자에게 전송
   - 또는 QR 코드로 제공 가능

2. **참여자가 링크를 클릭하면**
   - 자동으로 번호표가 발급됩니다
   - 본인의 번호와 현재 호출 번호를 확인할 수 있습니다

3. **참여자 화면 설명**
   ```
   ┌─────────────────────────┐
   │   📋 상담 대기실         │
   ├─────────────────────────┤
   │ 내 번호: 7번             │
   │ 현재 호출: 3번           │
   │ 대기 인원: 12명          │
   │                         │
   │ [새로고침]              │
   └─────────────────────────┘
   ```

#### 💡 팁
- 한 번 번호를 받으면 다시 접속해도 같은 번호가 유지됩니다
- 참여자가 실수로 창을 닫아도 다시 접속하면 번호를 확인할 수 있습니다
- 관리자가 번호를 변경하면 모든 참여자 화면이 자동으로 업데이트됩니다

## 자주 묻는 질문

### Q1. Excel 파일을 업로드했는데 오류가 나요
**A.** 다음을 확인해주세요:
- 파일 형식이 .xlsx 또는 .xls인지 확인
- 첫 번째 줄이 제목(이름, 전화번호, 사업단)으로 되어 있는지 확인
- 데이터가 빠진 칸이 없는지 확인
- 특수문자나 이상한 글자가 없는지 확인

### Q2. 지도에 장소가 안 나타나요
**A.** 주소를 확인해주세요:
- 정확한 도로명 주소 또는 지번 주소를 사용했는지 확인
- "서울시" 대신 "서울특별시"처럼 공식 명칭 사용
- 오타가 없는지 확인

### Q3. 참여자가 번호표를 못 받았다고 해요
**A.** 다음을 확인해주세요:
- 올바른 방 링크를 전달했는지 확인
- 참여자가 인터넷에 연결되어 있는지 확인
- 링크를 클릭하고 잠시 기다려보세요 (1-2초 소요)

### Q4. 같은 사람이 여러 번 번호를 받을 수 있나요?
**A.** 아니요, 한 사람은 한 방에서 하나의 번호만 받을 수 있습니다.
- 같은 기기(휴대폰)로 다시 접속하면 이전 번호가 표시됩니다
- 실수로 여러 번호를 받을 걱정이 없습니다

### Q5. 번호표 방을 삭제하고 싶어요
**A.** 현재는 방 삭제 기능이 없습니다.
- 방을 더 이상 사용하지 않으려면 참여자들에게 링크를 공유하지 않으면 됩니다
- 새로운 날에는 새로운 방을 만들어 사용하세요

### Q6. 여러 명의 관리자가 함께 관리할 수 있나요?
**A.** 네, 가능합니다.
- 여러 컴퓨터나 스마트폰에서 동시에 접속 가능
- 한 관리자가 번호를 변경하면 다른 관리자 화면도 자동 업데이트

### Q7. 인터넷이 끊기면 어떻게 되나요?
**A.**
- 참여자: 마지막으로 받은 정보가 그대로 표시됩니다
- 인터넷이 다시 연결되면 최신 정보로 자동 업데이트
- 발급받은 번호는 사라지지 않습니다

### Q8. Excel 파일 예시를 받을 수 있나요?
**A.** 예시 파일 형식:

**참여자 정보 파일 (member.xlsx)**
```
| 이름    | 전화번호       | 사업단      |
|---------|---------------|------------|
| 홍길동  | 010-1111-2222 | 공원관리단  |
| 김영희  | 010-3333-4444 | 환경미화단  |
```

**활동 장소 파일 (place.xlsx)**
```
| 사업단명     | 주소                              |
|-------------|-----------------------------------|
| 공원관리단   | 서울특별시 강남구 테헤란로 123     |
| 환경미화단   | 서울특별시 서초구 서초대로 456     |
```

## 용어 설명

### Excel (엑셀)
- Microsoft에서 만든 표 계산 프로그램
- 데이터를 표 형태로 정리할 때 사용
- 파일 확장자: .xlsx, .xls

### 업로드
- 컴퓨터나 스마트폰의 파일을 인터넷 시스템으로 올리는 것
- "파일 선택" 버튼을 클릭하면 자동으로 업로드됩니다

### 데이터베이스
- 시스템이 정보를 저장하는 창고
- "데이터베이스에 저장" 버튼을 누르면 정보가 시스템에 영구 보관됩니다

### 실시간
- 변경 사항이 즉시 반영되는 것
- 관리자가 번호를 변경하면 참여자 화면이 바로 업데이트

### 링크
- 특정 웹 페이지로 바로 이동할 수 있는 인터넷 주소
- 클릭하면 해당 페이지로 이동
