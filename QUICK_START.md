# 빠른 시작 가이드 (5분 완성!)

> **처음 사용하시나요?** 이 문서만 따라하시면 바로 시작할 수 있습니다!

---

## 🎯 1단계: 시스템 접속

### 컴퓨터 또는 스마트폰에서

1. 인터넷 브라우저(크롬, 엣지, 사파리 등) 실행
2. 주소창에 입력: `http://시스템주소:5173`
3. 메인 화면 확인 ✅

---

## 📝 2단계: 참여자 정보 등록 (첫 사용 시)

### Excel 파일 준비

먼저 Excel에서 이렇게 작성하세요:

```
┌──────────┬───────────────┬────────────┐
│ 이름     │ 전화번호      │ 사업단     │
├──────────┼───────────────┼────────────┤
│ 홍길동   │ 010-1234-5678 │ 공원관리단 │
│ 김영희   │ 010-2345-6789 │ 환경미화단 │
└──────────┴───────────────┴────────────┘
```

### 업로드하기

1. 메인 화면 → **"회원 Excel 업로드"** 클릭
2. **"엑셀 파일을 선택하세요"** 클릭 → 파일 선택
3. 화면에 표시된 정보 확인
4. **"데이터베이스에 저장"** 클릭
5. 완료! ✅

---

## 🎫 3단계: 번호표 시스템 사용

### 대기실 만들기 (관리자)

1. 메인 화면 → **"🔐 방 관리 (관리자)"** 클릭
2. 방 제목 입력 (예: "상담 대기실")
3. **"방 만들기"** 클릭
4. 완료! ✅

### 참여자에게 링크 공유

1. 만든 방의 **"링크 복사"** 버튼 클릭
2. 카카오톡이나 문자로 참여자에게 전송
3. 참여자가 링크를 클릭하면 자동으로 번호표 발급!

### 번호 호출하기

1. 해당 방의 **"입장"** 버튼 클릭
2. **"다음 번호 호출"** 버튼 클릭
3. 모든 참여자 화면에 실시간 반영! ✅

---

## 📍 4단계: 활동 장소 등록 (선택사항)

### Excel 파일 준비

```
┌──────────────┬──────────────────────────────────┐
│ 사업단명     │ 주소                             │
├──────────────┼──────────────────────────────────┤
│ 공원관리단   │ 서울특별시 강남구 테헤란로 123   │
│ 환경미화단   │ 서울특별시 서초구 서초대로 456   │
└──────────────┴──────────────────────────────────┘
```

### 업로드하기

1. 메인 화면 → **"Excel 관리 (지도)"** 클릭
2. 파일 선택 → 자동 업로드
3. 지도에서 위치 확인
4. **"데이터베이스에 저장"** 클릭
5. 완료! ✅

---

## ⚠️ 주의사항

### ✅ 이것만 기억하세요!

1. **Excel 파일**
   - 첫 번째 줄은 제목 (이름, 전화번호, 사업단)
   - 두 번째 줄부터 실제 데이터
   - 파일 형식: .xlsx 또는 .xls

2. **주소 입력**
   - 정확한 주소 사용 (도로명 또는 지번)
   - "서울특별시" 같은 공식 명칭 사용

3. **번호표 시스템**
   - 한 사람은 한 방에서 하나의 번호만 받음
   - 관리자가 번호 변경하면 모든 화면 자동 업데이트

## 🆘 문제 발생 시

### 자주 발생하는 문제

| 문제 | 해결 방법 |
|------|-----------|
| Excel 업로드 안 됨 | 파일 형식(.xlsx, .xls) 확인 |
| 지도에 장소 안 나타남 | 주소가 정확한지 확인 |
| 번호표 못 받음 | 인터넷 연결 확인, 링크 재전송 |
| 화면이 안 보임 | 새로고침(F5) 또는 페이지 다시 접속 |

### 그래도 안 되면?

**시스템 관리자에게 연락**하며 다음 정보를 알려주세요:
- 사용하던 기능 (예: "회원 Excel 업로드")
- 오류 메시지 (화면에 표시된 내용)
- 사용한 파일 정보

## 📱 스마트폰 사용 팁

### 참여자용 (번호표 확인)

1. 링크를 받으면 바로 클릭
2. 본인 번호와 현재 호출 번호 확인
3. 화면을 켜둔 상태로 대기
4. 번호가 바뀌면 자동으로 업데이트됨

### 관리자용

- 컴퓨터 사용 권장
- 스마트폰으로도 가능하나 화면이 작을 수 있음
- 태블릿(아이패드 등)이 가장 편리
